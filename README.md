## Getting started

Clone o repositório:
`git clone https://github.com/charleseduardome/challenge2-vulcanet-pecazap`

Na pasta raiz execute `yarn` para instalar todas as dependencias do projeto.

Na pasta raiz execute para iniciar a aplicação:

```sh
yarn start
ou
npm start
```
